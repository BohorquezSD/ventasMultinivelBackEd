Login: Prueba de conexion con la base de datos oracle, recibe user y pass

verPrivs: Muestra los privilegios de un usuario, recibe user, pass y el user a consultar(search)

verRepVentas: Muestra el representante a consultar recibe user, pass y el a consultar(search)

regCliente: Registra un usuario, recibe user, pass de quien registra,y los datos del cliente, retorna estado de aprovacion o error
